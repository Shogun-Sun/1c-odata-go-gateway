# Academic Booking API

Сервис для управления бронированием аудиторий учебного заведения. API выступает в роли связующего звена (middleware) между пользовательскими приложениями и системой 1С:Предприятие, обеспечивая работу с расписанием и занятостью помещений через протокол OData.

## Основные возможности

* **Управление бронированием:** Автоматизация процессов бронирования аудиторий.
* **Интеграция с 1С:** Безопасное проксирование данных из учетной системы 1С.
* **Контейнеризация:** Готовая к запуску инфраструктура в Docker (1С-сервер + API-сервис на Go).

## Начало работы

### Установка и запуск

1. **Склонируйте репозиторий:**
```bash
git clone https://github.com/Shogun-Sun/1c-odata-go-gateway.git
cd academic-booking-api
```

Или

```bash
git clone git@github.com:Shogun-Sun/1c-odata-go-gateway.git
cd academic-booking-api
```

2. **Настройка окружения:**
Скопируйте пример файла конфигурации и заполните данные для подключения:
```bash
cp .env.example .env
```

3. **Запуск:**
Соберите и запустите проект командой:
```bash
docker compose up -d --build
```
